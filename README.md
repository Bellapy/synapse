# Synapse - The Thought Weaver

Um Knowledge Graph dinâmico gerado por IA que transforma dúvidas abstratas em constelações de conhecimento visual.

**Stack:**
- **Frontend:** React (Vite), Tailwind CSS, React Force Graph, Zustand
- **Backend:** Python (FastAPI), LangChain, Pydantic
- **AI Engine:** Google Gemini
- **Database:** Supabase (PostgreSQL)
